VisualStudioProject

@Incubating interface VisualStudioProject : BuildableComponentSpec

A visual studio project, created from one or more org.gradle.nativeplatform.NativeBinary instances. The content and location of the generate project file can be modified by the supplied methods: 

 apply plugin: "visual-studio" model { visualStudio { projects.all { projectFile.location = "vs/${name}.vcxproj" projectFile.withXml { asNode().appendNode('PropertyGroup', [Label: 'Custom']) .appendNode('ProjectDetails', "Project is named ${project.name}") } } } }
