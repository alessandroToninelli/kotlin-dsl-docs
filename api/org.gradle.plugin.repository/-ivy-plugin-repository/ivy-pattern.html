ivyPattern

abstract fun ivyPattern(pattern: String): Unit

Adds an independent pattern that will be used to locate ivy files in this repository. If this pattern is not a fully-qualified URL, it will be interpreted as a file relative to the project directory. It is not interpreted relative the URL specified in #setUrl(Object). Patterns added in this way will be in addition to any layout-based patterns added via #setUrl.

Parameters

pattern - The ivy pattern.
