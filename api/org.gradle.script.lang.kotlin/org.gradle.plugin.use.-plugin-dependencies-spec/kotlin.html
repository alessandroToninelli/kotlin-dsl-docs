fun PluginDependenciesSpec.kotlin(module: String, version: String? = null): PluginDependencySpec

Builds the plugin dependency specification for the named Kotlin Gradle plugin module at the given version.

Visit the plugin portal to see the list of available plugins.

Parameters

module - simple name of the Kotlin Gradle plugin module, for example "jvm", "android", "kapt", "plugin.allopen" etc... version - optional desired version, null implies embeddedKotlinVersion.
