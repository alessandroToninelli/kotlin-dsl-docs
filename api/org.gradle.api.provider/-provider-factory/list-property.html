listProperty

abstract fun <T : Any> listProperty(elementType: Class<T>): ListProperty<T>

Creates a PropertyState implementation to hold a List of the given element type. The property with have an empty list as its initial value.

The implementation will return immutable List values from its query methods.

Parameters

elementType - The type of element.

- The type of element.

Exceptions

IllegalArgumentException - If the element type is a parameterized type.

Return
The property. Never returns null;
