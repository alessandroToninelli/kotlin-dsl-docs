removeTestOutputListener

open fun removeTestOutputListener(listener: TestOutputListener): Unit

Unregisters a test output listener with this task. This method will only remove listeners that were added by calling #addTestOutputListener(org.gradle.api.tasks.testing.TestOutputListener) on this task. If the listener was registered with Gradle using org.gradle.api.invocation.Gradle#addListener(Object) this method will not do anything. Instead, use .

Parameters

listener - The listener to remove.
