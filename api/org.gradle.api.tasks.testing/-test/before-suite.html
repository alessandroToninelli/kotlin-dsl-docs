beforeSuite

open fun beforeSuite(closure: Closure<Any>): Unit

Adds a closure to be notified before a test suite is executed. A org.gradle.api.tasks.testing.TestDescriptor instance is passed to the closure as a parameter.

This method is also called before any test suites are executed. The provided descriptor will have a null parent suite.

Parameters

closure - The closure to call.
