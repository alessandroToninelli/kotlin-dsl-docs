BuildCacheException

open class BuildCacheException : GradleException

BuildCacheException is the base class of all exceptions thrown by a BuildCacheService.

A BuildCacheException is considered a non-fatal failure. Gradle should still be able to complete a build even when a BuildCacheException is encountered.
